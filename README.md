# Gestion des Collaborateurs - SQLI

Ce projet est une **application web** développée dans le cadre de mon stage de fin d'études chez **SQLI**. L'objectif de cette application est de faciliter la gestion des collaborateurs, des formations et des certifications au sein de l'entreprise SQLI. Elle permet une gestion centralisée et automatisée des informations des collaborateurs.

## Technologies utilisées

- **Langage de programmation** : C#  
- **Framework** : ASP.NET Core, MVC  
- **Base de données** : SQL Server  
- **ORM** : Entity Framework Core  
- **Frontend** : HTML, CSS, JavaScript, Bootstrap  
- **IDE** : Visual Studio

## Fonctionnalités principales

- **Authentification** : Connexion via un formulaire de login sécurisé.
- **Gestion des collaborateurs** : Ajouter, modifier, supprimer et visualiser les informations des collaborateurs.
- **Affectation de formations et certifications** : Possibilité d’affecter des formations et certifications aux collaborateurs.
- **Rappels** : Alertes pour les dates importantes, telles que la fin de période d’essai ou l’évaluation annuelle.
- **Dashboard** : Vue globale avec statistiques sur les collaborateurs.

## Fonctionnalités détaillées

1. **Page de login**  
   Permet à l'utilisateur de se connecter avec ses identifiants.

2. **Gestion des collaborateurs**  
   - Ajouter un collaborateur à la base de données.  
   - Modifier ou supprimer les informations des collaborateurs existants.  
   - Afficher la liste complète des collaborateurs.

3. **Affectation de formations et certifications**  
   - Affecter des formations et certifications à un ou plusieurs collaborateurs.  
   - Visualiser et gérer les formations et certifications.

4. **Rappels de dates importantes**  
   Affichage des collaborateurs avec des périodes d'essai ou d'évaluation annuelle (BAP) imminentes.
